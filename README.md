# Предсказание спроса на такси
Данный проект - ремейк решения финала хакатона DigitalSuperHero с дополнениями. На хакатоне было необходимо 
рассчитать прогнозные значения ожидаемой продолжительности жизни на ближайшие 3 года в зависимости от факторов, 
влияющих на этот целевой показатель. Решение имело наилучший MAE = 0.35 на валидационной выборке (т.е. общее накопленное
отклонение за 4 года составило 3,5 месяца "в плюс"). 

Используемые тесты и метрики для проверки гипотез:
  - тест Дики-Фулера на единичный корень
  - тест Гренджера на причинность
  - коэффициент корреляции Пирсона
  - ранг матрицы
  
Используемые алгоритмы для расчёта прогнозных значений:
  - векторная авторегрессия
  - стохастический градиентный бустинг
